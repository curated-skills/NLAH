# Runtime Policy

This directory holds the standing runtime policy for LinguaClaw.

- `SKILL.md` is the imported seed charter from the research prototype.
- It is the default source of always-on runtime behavior.
- It should evolve more slowly than ordinary harness modules.

In the public architecture, runtime policy is different from a harness module:

- runtime policy defines the baseline interpreter behavior,
- harness modules add task or workflow patterns on top of that baseline.
